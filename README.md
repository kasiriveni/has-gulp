# has-gulp
Check if a project is using Gulp

## Install

```
$ npm install --save has-gulp
```

## Usage

```js
const hasGulp= require('has-gulp');
```


## API

### hasGulp([cwd])

Returns a `boolean`.

#### cwd

Type: `string`<br>
Default: `process.cwd()`

Current working directory.
